# PMOS_VI_characteristic
Purpose: To extract and plot the Output and Transfer characteristics of a Sky130 PMOS device.

PDK: Skywater 130nm (sky130_fd_pr__pfet_01v8).

Toolchain: Xschem (Schematic) and Ngspice (Simulation).

Key Feature: Demonstrates the correct SPICE .control syntax to negate the internal absolute current magnitude, ensuring plots align with standard 3rd-quadrant semiconductor physics conventions
